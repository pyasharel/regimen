# ASO Metadata Strategy for v1.0.3

## Final Recommendations

### Name (No Change)
```
Regimen: Peptide & TRT Tracker
```
- 30 characters, optimized
- Covers peptide + TRT audiences

### Subtitle (Update)
```
Ozempic & Mounjaro Calculator
```
- Targets high-volume GLP-1 brand searches
- Combined with name, covers all user segments
- Replaces: "Calculator, GLP1, & TRT" (had TRT duplication)

### Keywords (100 characters)
```
semaglutide,tirzepatide,ozempic,mounjaro,testosterone,trt,injection,peptide,bpc157,retatrutide,shotsy
```

**Keyword Strategy:**
- `semaglutide`, `tirzepatide`: Drug names (high intent)
- `ozempic`, `mounjaro`: Brand names (highest volume)
- `testosterone`, `trt`: TRT audience
- `injection`, `peptide`: Action + category
- `bpc157`, `retatrutide`: Niche compounds (high intent)
- `shotsy`: Competitor name (aggressive traffic capture)

### Words NOT in Keywords (already in Name/Subtitle)
- regimen, peptide, trt, tracker, calculator, ozempic, mounjaro

- Keywords can be updated with v1.0.3 submission
